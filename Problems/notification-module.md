**Design a notification module library**

**Functional** 
1. Receive notifications from backend through FirebaseMessagingService
2. Parse payload for different notification types
3. Persistence
4. Decide when to display notification
5. Present notifications
6. Action handling on the notifications, navigation, ack, etcs.
7. Check past notifications
8. Set notification settings, mute, hide...

**Non-functional**
1. Fast & responsive UI
2. No duplicate or missing notifications
3. Battery efficiency
4. privacy & security
5. Maintainability & scalability
